# UkrainianPolytopiaMod
![](https://img.shields.io/github/downloads/johnklipi/UkranianPolytopiaMod/total)

## Custom Ukrainian translation for the game The Battle of Polytopia.
### Installation instructions

Download PolyMod Installer from https://polymod.dev/ and install PolyMod.

Download one of our releases https://github.com/johnklipi/UkranianPolytopiaMod/releases/latest, and then put it to the `Mods` folder in **the Polytopia root folder**.

Open the game settings, and in language list select Українська.

Enjoy!


## Кастомний український переклад для гри The Battle of Polytopia.
### Інструкція з встановлення

Завантажте інсталятор PolyMod з сайту https://polymod.dev/ та встановіть PolyMod.

Завантажте один із наших релізів https://github.com/johnklipi/UkranianPolytopiaMod/releases/latest, а потім помістіть його в папку `Mods` у **кореневій папці Polytopia**.

Відкрийте налаштування гри та у списку мов виберіть «Українська».

Приємної гри!
